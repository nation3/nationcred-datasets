# Discourse

https://forum.nation3.org/

## Datasets

- `output/discourse-usernames.csv`

  Contains the link between a Discourse username and an Ethereum address.

## Install

```
npm install
```

## Lint

```
npm run lint
```

## Build & Run

### Build

```
npm run build
```

### Generate Usernames CSV

```
npm run generate-csv
```
